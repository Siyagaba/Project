# Metadata Health Report Generator

## 📌 Overview
This project generates a simple metadata health report and displays it as an HTML file. It helps visualize data quality, stale assets, and anomalies in a structured format.

## 🚀 Features
- Generates metadata report using Python
- Saves report as an HTML file
- Simple and easy-to-run prototype
- Optional email feature (can be enabled)

## 🛠️ Tech Stack
- Python
- HTML (for report UI)

## ▶️ How to Run
1. Install Python
2. Run the script:
   python app.py
3. Open generated `report.html` in browser

## 📂 Output
- report.html (generated file)
